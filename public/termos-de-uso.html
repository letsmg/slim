📋 Termos de Uso

Última atualização: 16 de maio de 2026

Bem-vindo ao Gerenciador de Frotas. Ao utilizar este sistema, você concorda com os termos descritos abaixo. Caso não concorde com algum dos termos, recomendamos que não utilize a plataforma.

1. Coleta de Dados

Para garantir a segurança, integridade e melhoria contínua do sistema, coletamos automaticamente os seguintes dados durante sua navegação:

  • Endereço de IP - Identificador único do seu dispositivo na rede
  • Localização geográfica aproximada (GPS) - Coordenadas de latitude e longitude obtidas via geolocalização do navegador
  • Dados de sessão - Informações de navegação necessárias para o funcionamento do sistema

2. Armazenamento e Proteção de Dados

🔒 Compromisso com sua privacidade:

Todos os dados de IP e localização GPS são armazenados exclusivamente sob a forma de hash criptográfico Argon2id, o padrão mais seguro disponível atualmente. Isso significa que:

  • Seus dados são irreversíveis - não podemos recuperar seu IP ou localização original
  • O hash é único por sessão e não pode ser vinculado a outros sistemas
  • Não compartilhamos estes dados com terceiros
  • Os dados são utilizados exclusivamente para aprimoramento do sistema

3. Finalidade da Coleta

Os dados coletados são utilizados para:

  • ✅ Melhorar a experiência do usuário no sistema
  • ✅ Identificar padrões de uso para otimização de desempenho
  • ✅ Prevenir e detectar atividades fraudulentas ou maliciosas
  • ✅ Gerar estatísticas anônimas de acesso
  • ✅ Aprimorar a segurança geral da plataforma

4. Upload de Imagens

Ao fazer upload de imagens no sistema (fotos de veículos, manutenções, etc.), você declara ser o proprietário ou ter autorização para uso das mesmas. Todas as imagens passam por verificação automática de conteúdo para garantir a conformidade com as políticas de uso aceitável.

5. Direitos do Usuário

Você tem o direito de:

  • Solicitar esclarecimentos sobre o tratamento dos seus dados
  • Solicitar a exclusão dos seus dados armazenados
  • Revogar o consentimento a qualquer momento

6. Alterações nos Termos

Estes termos podem ser atualizados periodicamente. Recomendamos a revisão regular desta página. O uso continuado do sistema após alterações constitui aceitação dos novos termos.

7. Contato

Para questões relacionadas a estes termos ou ao tratamento de dados, entre em contato através dos canais oficiais do sistema.
